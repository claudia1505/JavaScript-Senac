OnClick

Para aparecer um botão, clicando nele apenas uma vez. Clique para acionar a função que produzirá "Olá Mundo":

ondblclick​

O onclick aciona uma função quando um elemento é clicado 2 vezes.

onmouseover​​​

O onmouseover​​ aciona uma função que um elemento so aparece se depois que passar o mouse em cima dele.

onmouseout​​​

Para aparecer uo elemento quando passarmos o mouse e tirarmos o mouse dentro do imput.


onmouseup​​​
